# SwiftXcodeSnippets
Swift Code Snippets added to my Xcode Code Snippet Library

```bash
cp *.codesnippet ~/Library/Developer/Xcode/UserData/CodeSnippets/
```

```swift
operator - operator overload
```

```swift
singleton - singleton
```

```swift

```
